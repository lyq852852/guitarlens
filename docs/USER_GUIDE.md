# GuitarLens v0.5.0 用户指南

## 简介

GuitarLens 是一款 AI 驱动的吉他扒带软件，能够自动分析音频文件，识别吉他音符和弦，并生成可编辑的谱面。

---

## 安装

### macOS

1. 下载 `GuitarLens-0.5.0.dmg`
2. 双击打开，将 GuitarLens 拖入 Applications
3. 首次运行右键点击 → 打开（绕过 Gatekeeper）

### Windows

1. 下载 `GuitarLens-0.5.0-Setup.exe`
2. 双击运行安装程序，按提示完成安装
3. 从开始菜单或桌面快捷方式启动

### 首次启动

首次启动会自动下载 AI 模型文件（约 1.5 GB），请保持网络连接，等待下载完成。

---

## 快速开始

### 第一步：导入音频

- 点击主界面的 **"Upload Audio"** 按钮
- 或直接将音频文件**拖拽**到应用窗口
- 支持格式：MP3、WAV、FLAC、AAC、OGG

### 第二步：处理音频

点击 **"Process Audio"** 按钮，等待 AI 分析完成。

进度条会显示当前处理阶段：
| 阶段 | 说明 |
|------|------|
| 0–30% | 音轨分离（Demucs） |
| 30–60% | 音符识别（basic-pitch） |
| 60–80% | 和弦识别 |
| 80–100% | 谱面生成 |

### 第三步：查看谱面

处理完成后，谱面自动显示。顶部工具栏可切换视图：

| 按钮 | 视图 |
|------|------|
| Tab | 六线谱（默认） |
| Staff | 五线谱 |
| Both | 六线谱 + 五线谱 |
| Chords | 和弦图 |

### 第四步：播放和跟弹

点击播放器的 **▶ 播放** 按钮，谱面会自动高亮当前音符，方便跟弹练习。

### 第五步：编辑（可选）

点击工具栏 **✏️ Edit** 进入编辑模式：
- 点击音符可修改品位/弦
- 右键音符可删除
- 点击空白处可添加音符
- `Cmd/Ctrl + Z` 撤销，`Cmd/Ctrl + Y` 重做

### 第六步：导出

点击 **⬇ Export** 选择导出格式：
- **PDF** — 可打印的谱面
- **MusicXML** — 兼容 MuseScore、Finale 等
- **GuitarPro (.gpx)** — 兼容 GuitarPro 软件

## 支持的和弦类型

GuitarLens 可识别以下和弦：

| 类型 | 示例 |
|------|------|
| 大三和弦 | C, D, E, F, G, A, B |
| 小三和弦 | Cm, Dm, Em, Am, Bm |
| 属七和弦 | G7, D7, A7 |
| 大七和弦 | Cmaj7, Fmaj7 |
| 小七和弦 | Am7, Em7, Dm7 |
| 挂留和弦 | Csus2, Dsus4 |
| 加音和弦 | Cadd9, Gadd9 |
| 转位和弦 | C/E, G/B |

| 快捷键 | 功能 |
|--------|------|

## 常见问题

**Q: 处理速度很慢？**  
A: 首次处理需要加载 AI 模型（约 30 秒），后续处理会更快。长音频（>5 分钟）处理时间约 2–5 分钟。

**Q: 识别结果不准确？**  
A: 效果最佳的输入是干净的吉他录音。混音复杂的歌曲识别准确率会降低。可在编辑模式下手动修正。

**Q: 支持贝斯吉他吗？**  
A: 目前主要针对六弦吉他优化，贝斯识别功能在 v0.6.0 中加入。

**Q: 如何备份项目？**  
A: 项目文件保存在 `~/Documents/GuitarLens/` 目录，直接复制该文件夹即可备份。

**Q: 可以在 Windows 和 Mac 之间共享项目吗？**  
A: 可以。项目文件（`.glp`）和导出文件（PDF/MusicXML/GPX）均为跨平台格式。
