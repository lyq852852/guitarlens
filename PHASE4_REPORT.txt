🚀 快速开始
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

   npm install

2. 启动开发:
   npm start

3. 使用编辑器:
   - 上传音频文件
   - 点击 "Process Audio"
   - 编辑谱面
   - 撤销/重做
   - 保存项目
   - 导出文件
