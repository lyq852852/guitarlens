🔧 技术实现

音符转 Tab 算法:
  • 标准调弦（E A D G B E）
  • 频率 → MIDI 转换
  • 最优弦选择（优先低品位）
  • 时值计算（相对于 BPM）

和弦识别:
  • 15+ 种和弦类型
  • 音程匹配算法
  • 自动指位计算
  • 和弦图库

谱面格式:
  • alphaTab（用于渲染）
  • GuitarPro XML（用于导出）
  • MusicXML（用于导出）

🚀 快速开始
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

1. 安装依赖:
   npm install

2. 启动开发:
   npm start

3. 使用谱面查看器:
   - 上传音频文件
   - 点击 "Process Audio"
   - 查看谱面（Tab/Staff/Chords）
   - 切换视图
   - 导出文件
